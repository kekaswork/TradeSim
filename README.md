# TradeSim

A simple trading training platform where users can practice trading stocks and crypto using virtual money.  
Backend uses Go microservices, frontend uses React + TypeScript, and PostgreSQL is the primary database.

## Features
- User registration and authentication  
- Virtual wallet with initial balance  
- Real-time market data (stocks & crypto)  
- Buy/Sell simulation  
- Portfolio and trade history  
- Event-driven architecture (Kafka/RabbitMQ)  

## Tech Stack
- Go (microservices, gRPC + REST)
- React + TypeScript
- PostgreSQL
- Kafka or RabbitMQ
- Docker
